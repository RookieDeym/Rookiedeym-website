Dropdown MENU

Its compulsary to use the SELECT element to create a drop down list

The OPTION element is used inside a SELECT tag to add choices for the user

<select> <option> Option 1 </option> <option> Option 2 </option> <option> Option 3 </option> </select>

RESULT ↓





You can use the SELECT element as a part of a FORM to collect user input

<form> <select> <option> Option 1 </option> <option> Option 2 </option> </select> </form>

Submitting a form needs information to a database. The NAME attribute is needed to submit the form data

<form> <select name="color"> <option> Red </option> <option> Blue </option> </select> </form>

Data will be sent when the form is submitted. You can control the data each option sends with the value attribute

<form> <label for="dropdown">Choose an option:</label> <select id="dropdown"> <option value="r">Red</option> <option value="g">Green</option> <option value="b">Blue</option> </select> <input type="submit"> </form>

You can also add labels to drop down menu

<form> <label>Vehicle Type:</label> <select name="Vehicle"> <option value="car">Car</option> <option value="bus">Bus</option> </select> </form>

Labels and drop-down menus are connected with for and id attributes, just like any other form element

<form> <label for="s1">Vehicle Type:</label> <select name="vehicle" id="s1"> <option value="car">Car</option> <option value="bus">Bus</option> </select> </form>

Correctly connecting labels and form fields will increase hit area and improve accessibility.


        label = For Attribute
        select = id Attribute

The Select element can be used in forms in combination with input elements

<input type="text"> : text box <input type="checkbox"> : checkbox <input type="radio"> : radio button <input type="submit"> : submit button 
     
        •The select tag is used to create drop-down menu.
        •The option tag is used to give choices to the user.
        •You can use drop-down menus in forms, in combination with other input elements.

Video Tags

This will show you how to add videos on your webpage

Example:

<video> <source src="file.mp4" type="video/mp4"> </video>

The TYPE attribute specifies the format of the video file



Here are the other example, if you want to add multiple videos

<video> <source src="file.mp4" type="video/mp4"> <source src="file.webm" type="video/webm"> </video>

Btw if you include multiple options, the browser will choose the first one it supports. BUT if all of the options are supported it'll display all of the options from top to bottom.



You can also add text between the video tags but the text will only display in a browser if it doesnt support the video element.

Here's the example:

<video> <source src="videofile.mp4" type="video/mp4"> <source src="videofile.webm" type="video/webm"> Video not supported </video>

You can also display play/pause, volume and other video controls with the controls attribute. Ofcourse its optional :) But if you dont include control attribute the video wont play, so the only thing that you can do is put the autoplay muted

<video controls> <source src="file.mp4" type="video/mp4"> Video not supported </video> 
        •Embeds video element     :  video
        •File and format options  :  source
        •File location            :  src attribute
        •Format information       :  type attribute


Audio Container Tag

<audio> <!--audio file--> </audio>

Just like the video, the source tag is used to add options for audio, src attribute to adds the audio file url, the type attribute adds the format

<audio> <source src="file.mp3" type="audio/mpeg"> <source src="file.ogg" type="audio/ogg"> </audio>

Just like video, you can add audio files in different formats. Common audio formats are: MP3, OGG, and WAV

Atributes usually come in the name='value' format. Why is the controls attribute an exception? Because IT HAS A DEFAULT VALUE IF NO OTHER IS SPECIFIED


You can use autoplay, muted and loop attributes to control the behavior of the multimedia element. Just like controls, these attributes have their default values omitted.

        •No sound                      : muted        
        •Starts playing automatically  : autoplay    
        •Plays in repeat               : loop
        •Shows media controls          : controls
LAYOUT OF YOUR WEBPAGE

Header, Main, Footer

•The Header container tag usually contains introductory information. It often contains navigation links, a menu and/or a search bar. Brand elements like logos are usually found in the header too.

<header> <!--Introductory information--> </header>

•The Main container tag is used to include the main content of a web page

<main> <!--Main content--> </main>

•The Footer container tag is often contains contact and legal information and links.

<footer> <!--Footer information--> </footer>

Reminder: Header, Main and Footer elements are nested inside the body container tag

<b0dy> <header> <!--Intro info--> </header> <main> <!--Main Content--> </main> <footer> <!--Footer info--> </footer> </b0dy>
STRUCTURE OF WEBPAGES

Article represents an independent, self-contained piece of content

<article> <h1>5 Future-Proof jobs</h1> <p>Every industry needs coders</p> </article>

Section helps to breakdown the content into parts. It is a Semantic tag. It usually includes a heading.

<section> <h2>About Us</h2> <p>We are a small company...</p> </section>

Aside is used for secondary, aditional or somehow related content

<aside> <h3>Related topics</h3> <p>Did you know...</p> </aside>

STYLE ATTRIBUTE

With the Style attribute, you can full customize the visual presentation of your websites, from typography and colors to layout more.

The Style attribute can be used to define properties like color, font size and alignment.

You can customize multiple properties at the same time. Just seperate each property : value pair with a semicolon (;)

<h1 style="color: blue; text-align: center; font-size: xx-large;">Centered Blue Heading</h1>

The background-color property is useful for styling elements like buttons

<button style="background-color: green;">OK</button>
